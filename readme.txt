---HOW TO PUSH ON BRANCH MASTER_--------

-- first pull the repo
git pull "repo-link"

--change the origin to the repo
git remote add origin "repo-link"

--add, commit etc.

--ex the main branch is called main
-- to switch to it use 
git checkout -b main

--for push
git push origin main
